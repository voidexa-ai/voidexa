## 3) Balance Notes

| Mechanic | Costing guidance |
|---|---|
| System Reboot | +1 cost over baseline. Avoid pairing with Hardened Core. |
| Cold Boot | -1 cost or bonus ETB. Good downside for efficient statlines. |
| Hot Deploy | +1 cost or lower stats. Dangerous on burst archetypes. |
| Priority Fire | Fine at low-mid cost. Best on precision units. |
| Cascade | Best on 4+ cost bruisers. Avoid on cheap swarm. |
| High Orbit | Evasion premium. Needs healthy Tracking Lock density. |
| Tracking Lock | Low-cost safety valve. Should appear more often than High Orbit. |
| Critical Breach | Strong premium. Put on low-attack assassins. |
| Auto-Repair Protocol | Defensive premium. Once per turn only. |
| Hull Siphon | Snowball risk. Best on modest attack values. |
| Persistent Systems | Mid-level premium. Great on support drones. |
| Stealth Coating | High utility. Keep away from top-end bombs. |
| Hardened Core | Very high premium. Slow expensive finishers only. |
| Emergency Launch | Tactical premium. Best on defenders and ambushers. |
| Twin Barrels | Very high damage premium. Fragile mid-cost attackers. |
