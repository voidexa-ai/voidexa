## Format note

The `.png` extension on `deep_space_01.png` matches the spacespheremaps source file (no transcoding loss). `deep_space_universe.png` is also PNG to preserve AI-generation fidelity; lossy JPG would risk introducing color artifacts in the dark nebula gradients.
